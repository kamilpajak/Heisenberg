# Installation

## Requirements

- Python 3.11 or higher
- An Anthropic API key (for AI analysis)

## Install via pip

```bash
pip install heisenberg
```

## Install from source

```bash
git clone https://github.com/kamilpajak/Heisenberg.git
cd Heisenberg
pip install -e .
```

## Environment Variables

Set your API key:

```bash
export ANTHROPIC_API_KEY="your-api-key"
```
